# Obsidian Conventions (for Rafa)

When editing context files, follow these rules so Obsidian renders nicely:

- **Wiki-links:** Use `[[filename]]` to link between files (Obsidian resolves these)
- **Tags:** Use `#tag` format for categorisation
- **Frontmatter:** Optional YAML at top of files for metadata
- **No broken links:** If referencing another file, make sure it exists
- **File names:** Lowercase with hyphens (already doing this)
